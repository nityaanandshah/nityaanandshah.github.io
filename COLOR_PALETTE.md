## ğŸ¨ Brightness & Saturation Variations

### Brightness Adjustments

For creating variations, adjust the **Lightness** value:

| Variation | Light Mode L% | Dark Mode L% | Use Case |
|-----------|---------------|--------------|----------|
| Darkest | -20% | +15% | Active states, emphasis |
| Dark | -10% | +10% | Hover states |
| Normal | Base | Base | Default state |
| Light | +5% | -5% | Hover on dark elements |
| Lightest | +10% | -10% | Disabled, inactive |

### Saturation Adjustments

For creating variations, adjust the **Saturation** value:

| Variation | Saturation | Use Case |
|-----------|------------|----------|
| Muted | -40% to -50% | Backgrounds, disabled states |
| Subtle | -20% to -30% | Secondary elements |
| Normal | Base | Default state |
| Vibrant | +10% to +15% | Hover, focus states |
| Maximum | 75-85% | Alerts, important CTAs |
